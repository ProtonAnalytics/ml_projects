# Air Medical Transport Demand Forecasting — ML Engineering Portfolio Project

This project demonstrates an end-to-end machine learning workflow for forecasting daily air medical transport demand. The use case is modeled around emergency medical aviation operations, where accurate demand forecasting can support aircraft readiness, staffing, dispatch planning, and regional operational awareness.

The workflow includes synthetic data generation, feature engineering, regression modeling, model evaluation, MLflow experiment tracking, and production-oriented batch scoring design.
